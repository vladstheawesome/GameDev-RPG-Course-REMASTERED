How to change the ambient color of trees:

"Skybox" Environment Lighting in the scene  does not affect the trees shader.
If you want to make the trees brighter or darker   Window>Lighting>Settings and select -gradient or -color instead -skybox in environment lighting source. Select the color you want.


How to enable camera effects:
1 Import "Post-Processing Stack"
2 Add to camera component Effects>Post Processing Behaviour
3 Drag&Drop "Post-Processing Profile" to Post Processing Behaviour
